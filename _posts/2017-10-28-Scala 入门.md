Scala 中 `val` 是不可变化的

```scala
scala> a = "Hello World"
<console>:25: error: reassignment to val
       a = "Hello World"
         ^
```

可以创建 `var` 从而可变

```scala
scala> var n = 3
n: Int = 3

scala> n += 1

scala> n
res1: Int = 4
```

Scala中可以指定数据类型

```scala
scala> var a:String = "hello"
a: String = hello
```
字符串拼接可以直接使用 `+`

```scala
scala> val a = "Hello"
a: String = Hello

scala> val b = a + "World"
b: String = HelloWorld
```

Scala 中可以多赋值

```scala
scala> val a,b = Array(1,2,3)
a: Array[Int] = Array(1, 2, 3)

scala> a(0)=5

scala> a
res5: Array[Int] = Array(5, 2, 3)

```
